# Micro Rogue
Minimalist roguelike inspired by Oscar Toledo's Boot Rogue (510 bytes)

# Features
 - Procedurally generated dungeon, 9 rooms, tic-tac-toe layout
 - Dark rooms
 - Static monsters A-Z (yet it's hard to beat the game!)
 - Weapon & armor items to increase attack/defense skills
 - User defined level of the Amulet of Yendor
 - Extensive & transparent stats

# Controls

    h, j, k, l - orthogonal movements (upper case to run)
    y, b, u, n - diagonal movements (upper case to run)
          r, e - fight monster / escape fight
             w - wait 50 turns (to regain HP)
             S - save game (game would be loaded from "rogue.json" if available)
